## M-Flix

M-Flix is a web application built using the famous and well-known JavaScript library React. This is a movie review app, and due to the use of React, the user interface is smooth, providing a user-friendly experience. The app showcases movie reviews, including their description, rating, availability, release date, and many other details. Please check the feature section below for more information.

## Features

- **Feature 1:**
  When you first enter the page, you can see a banner with a movie image showing the current top-rated movie. There, you can also find a "more" button; by clicking on it, you'll be taken to the detailed page of the movie, where you can see information like the description, release date, etc.

- **Feature 2:**
  As you scroll down, you'll find three sections with popular, top-rated, and upcoming movies. Clicking on an image allows you to check the details of the movie, and you can scroll left to explore more movies.

- **Feature 3:**
  At the top left corner, you can see a search bar where you can search for movies by entering the correct name. After clicking the search button, the movie will be displayed, and clicking on it will show you the details.

- **Feature 4:**
  In the top left corner, there's a hamburger icon. Clicking on it opens a drawer where you can find six options. The first three sections are the same as those encountered before: top-rated, popular, and upcoming movies. By clicking on them, you can access more movies based on your needs.

- **Feature 5:**
  In the menu bar, the genre section is interesting. By clicking on it, a genre page will open, allowing you to select which genre of movie you want, and it will show results based on your selection.

- **Feature 6:**
  In the menu bar, there's a "More" section that includes a "My Favourite" section. If you have visited the details page of any movie, there was an "Add to Favourite" button. Clicking it will show the movie on this page.

## Tech Stack

- JavaScript
- React
- Front-end tools: React Query, React Router DOM, Sass
- Movie API: TMDB
