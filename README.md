## M-Flix

 M-Flix is an web application made from famous and well known javascript libery React. This is an movie review app and due to the use of react the User Interface is smooth to use and user friendly experience.This app shows the reviews of the Movies its description its rating and availbility release date and many other things please check the feature section below for more imformation.

## Features

- Feature 1:
  When First enter the page you can see an banner with a movie image shown its the current top rated movie. there you can also see a more buuton by clicking that it will take detail page of the movie were you see the details like discription,release date etc
- Feature 2:
  if you scrool down you 3 section with popular,top rated and upcomming movies. You can click on the image and then check details of the movie. you can scrool left to check more movies 
- Feature 3:
  At the top Left corner you can see search bar were you can search movies by enter the currect name of the movie. after clicking on serach button it will show the movie and by clicking on it you can check details of the movie
- Feature 4:
  At the top left corner you can see a hamburger icon click on it and a drawer will open you can see 6 options. the first 3 section same section we encounterd before top rated,popular and upcoming movies by clicling them you can access more movies based on your neeed.
- Feature 5:
   In the menu bar the genre section is an interestin section by click on it a genre page will open there you can select which genre movie you want it will show results based on it
- Feeture 6:
  In the menu bar there is more section that My Favourite section if you havve visited the details page any movie there was a add favourite button present there if you click it the movie will show on this page 

## Tech Stack

- JavaScript
- React
- front-end-tools: React Query , React router dom , Sass
- Movie API: TMDB
